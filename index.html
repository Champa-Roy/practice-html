Five Important HTML Tags (with examples)

<h1> — Heading

This is a top-level heading. Use headings to structure content.

<h1>This is a heading</h1>

<p> — Paragraph

Paragraphs contain blocks of text.

<p>This is a paragraph.</p>

<a> — Anchor / Link

Links allow navigation to other pages or resources: Visit example.com

<a href="https://example.com">Visit example.com</a>

<img> — Image

Images display pictures. Always include a meaningful alt attribute.

<img src="image.jpg" alt="description">

<ul> <li> — Unordered List

Lists group related items.

  • First item
  • Second item
  • Third item
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
